# Portafolio Fotográfico de Rony Vivas

Este es tu nuevo sitio web profesional, diseñado con un estilo moderno, minimalista y elegante ("Dark Mode") para resaltar tu trabajo fotográfico.

## Estructura de Archivos
- **index.html**: El contenido principal de la página.
- **styles.css**: Todos los estilos visuales, animaciones y colores.
- **script.js**: Efectos de interacción y animaciones al hacer scroll.
- **assets/**: Carpeta con las imágenes y tu logo.

## Cómo personalizarlo
1. **Logo**: Ya he incluido tu logo (`logo-white.png`).
2. **Imágenes**:
   - Reemplaza las imágenes en la carpeta `assets/` con tus propias fotos.
   - `hero.png`: Imagen principal de fondo (formato horizontal, alta calidad).
   - `gallery1.png`, `gallery2.png`: Imágenes de la galería.
3. **Texto**:
   - Abre `index.html` con cualquier editor de texto para cambiar los textos de "Sobre Mí" y tu información de contacto.

## Cómo ver la página
Simplemente haz doble clic en el archivo `index.html` para abrirlo en tu navegador.
